# SQL-Food-Delivery-Analysis
SQL-based food delivery data analysis project focused on restaurant performance, delivery efficiency, pricing insights, and customer rating trends using MySQL

## Project Overview

This project focuses on analyzing a food delivery dataset using SQL to derive business insights related to restaurant performance, customer ratings, pricing trends, food preferences, and delivery efficiency.

The analysis was performed using MySQL and includes data cleaning, exploratory data analysis (EDA), business insights generation, subqueries, CASE statements, and window functions.

## Tools & Technologies Used

* MySQL
* SQL
* Excel

## Dataset Information

The dataset contains food delivery platform data with the following columns:

* ID
* Area
* City
* Restaurant
* Price
* Avg Ratings
* Total Ratings
* Food Type
* Address
* Delivery Time

## SQL Concepts Used

* SELECT Statements
* WHERE Clause
* GROUP BY
* HAVING
* ORDER BY
* Aggregate Functions (COUNT, AVG, SUM)
* CASE Statements
* Subqueries
* Window Functions (RANK)
* Data Cleaning using UPDATE and TRIM

## Business Problems Solved

### Restaurant Performance Analysis

* Identified top-rated restaurants with high customer engagement
* Ranked restaurants based on pricing

### City-wise Analysis

* Analyzed cities with the highest number of restaurants
* Compared average delivery times across cities

### Customer Preference Trends

* Identified highly rated food categories
* Analyzed restaurant distribution by food type

### Pricing Insights

* Categorized restaurants into Budget, Mid-Range, and Premium segments
* Identified restaurants with above-average pricing

### Data Cleaning

* Replaced null and blank values in the Food Type column
* Handled inconsistent data using SQL functions

## Key Insights

* Kolkata had the highest number of restaurants in the dataset
* Ahmedabad showed the fastest average delivery time
* Premium restaurants dominated higher pricing segments
* Certain food categories consistently received higher customer ratings
